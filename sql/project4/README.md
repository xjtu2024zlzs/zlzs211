# 课题四 SQL 提交说明

本目录用于提交课题四相对于主课题系统新增的数据库脚本。

## 文件说明

- `project4_data.sql`：课题四增量数据库脚本，包含数据集信息、数据文件管理、预处理样本、样本增强、特征融合、故障诊断和根因分析相关业务表结构及演示数据。

## 导入说明

请先由主课题导入 RuoYi 基础数据库 SQL，再导入本目录下的课题四增量 SQL。

推荐导入顺序：

1. 主课题 RuoYi 基础 SQL
2. `project4_data.sql`

## 注意事项

- 本目录不重复提交主课题基础 SQL。
- SQL 文件默认导入到 `ry-cloud` 数据库。
- 本脚本不包含数据库账号、密码和服务器地址等敏感信息。
