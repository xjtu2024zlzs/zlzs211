# 课题四 SQL 提交说明

本目录用于提交课题四相关数据库 SQL 文件，包括菜单 SQL、业务表 SQL 和初始化数据 SQL。

## 文件说明

- `menu.sql`：课题四菜单、页面路由、按钮权限相关 SQL。
- `table.sql`：课题四新增业务表结构 SQL。
- `init_data.sql`：课题四演示数据和初始化数据 SQL。

## 导入说明

请先导入 RuoYi 基础数据库 SQL，再导入本目录下的课题四 SQL 文件。

推荐导入顺序：

1. `menu.sql`
2. `table.sql`
3. `init_data.sql`

## 注意事项

- SQL 文件应导入到 `ry-cloud` 数据库。
- 表名建议使用课题四相关前缀，避免与其他课题表冲突。
- 提交前请确认 SQL 中不包含个人数据库密码、服务器地址等敏感信息。
